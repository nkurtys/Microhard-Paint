## Instructions to build the project
1. Download .cpp .h and .pro files into a folder
1. Build Project with qmake and .pro file
1. execute with make.exe

- Windows 10
- Qt Creator 4.12.4
- Based on Qt 5.14.2 (MSVC 2017, 32 bit)
